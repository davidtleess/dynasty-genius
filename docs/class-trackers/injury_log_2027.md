# 2027 Class — Injury Log

## Purpose

Tracks reported injuries, surgeries, and durability concerns for 2027 NFL Draft prospects. Used by:
- Rookie Evaluator (durability flag in Engine A feature set)
- Trade Calculator (low_sample_holdout / confidence reduction on injured prospects)
- Class tracker maintenance (factual gap-filling for `2027.md`)

Per Framework Rule 1 (Verify Status), only **reported and sourced** injuries are logged here. Speculation, "looked off" tape commentary, and unattributed Twitter reports are excluded.

## Logging Format

| Date | Player | Pos | School | Event | Severity | Source | Status |
|---|---|---|---|---|---|---|---|

**Severity scale:**
- `minor` — day-to-day, no surgery, no missed games projected
- `moderate` — multi-week absence, no surgery
- `major` — surgery required, multi-month absence
- `career-flag` — recurring or structural concern (e.g. multiple ACLs, chronic shoulder)

**Status values:** `active`, `resolved`, `lingering`, `archived`
