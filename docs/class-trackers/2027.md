# 2027 NFL Draft Class Tracker

## Purpose

Ground-truth reference for the 2027 NFL Draft class. Used by:
- Pick valuation in the Trade Calculator (2027 1st appreciation modeling)
- Rookie Evaluator (BPA + Needs-Based modes, post-2027 draft)

Per Framework Rule 1 (Verify Status), this document is **factual snapshot only**. Subjective hype, mock-draft slotting beyond top-tier consensus, and analyst opinion are excluded — those belong in `analyst_notes_2027.md`.

## Class-Level Snapshot (May 2026)

**Consensus shape:** WR-heavy at the top tier, with a competitive QB tier and a smaller but credible RB tier. Multiple mainstream outlets (PFF, ESPN, Draftsharks, SI) project the 2027 WR group as the best since the 2024 class, with comparisons drawn to 2022 (Wilson/Olave/London/Watson tier).

**What is *factually true* as of May 2026:**
- The 2026 college season has not yet started. All evaluations below are pre-junior-season and subject to material revision.
- The 2027 draft order is fully unknown. Pick value modeling must use estimated slot ranges, not assumed slots.

**What is *consensus* but not yet verified outcome:**
- Jeremiah Smith is the consensus WR1 and a top-3 overall projected pick across mainstream boards.
- Julian Sayin (Ohio State QB), Arch Manning (Texas QB), Dante Moore (Oregon QB), and Drew Mestemaker have been named in early QB consideration. None has yet established separation. Treat the QB tier as **unresolved** until late 2026.

## 2027 1st-Round Pick Value Implications — Superflex

This is the part that matters for the Trade Calculator.

**A WR-heavy headline class affects SF pick value asymmetrically across the round:**

| Pick Range | Effect of WR-Heavy Class | Reason |
|---|---|---|
| **1.01–1.04** | **Elevated** | Generational WRs (Smith tier) are RB-position-equivalent in dynasty value over a 6–8 year window. SF managers will pay top-of-round premium for a Smith-caliber WR1 anchor. |
| **1.05–1.08** | **Roughly stable to slightly elevated** | Class depth (per PFF/Draftsharks early rankings) suggests credible WR2/WR3-of-class talent extends into the back half. Reduces "wasteland" risk. |
| **1.09–1.12** | **Conditional — depends on QB tier resolution** | If Sayin/Manning/Moore separate by Nov 2026, late-1st value rises (a QB1 asset). If QB tier flattens, late-1st becomes a WR3-of-class or RB1-of-class — still useful but not premium. |

**The asymmetry that matters for trade valuation:**
A WR-heavy class **raises the variance** of late-1st outcomes in Superflex. In a QB-heavy class, every late-1st is at least a backup QB option. In a WR-heavy class, late-1st value depends on whether a SF-relevant QB is still on the board. **Trade Calculator should widen the confidence interval on 2027 mid-to-late 1sts until the QB tier resolves.**

**Implication for the "appreciating asset" thesis used in trade pitches:**
Still holds for early-to-mid 2027 1sts. Weakens for late 2027 1sts if QB tier underperforms 2026 college season. Revisit this section monthly during college season.
